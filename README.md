## 簡述
類型：跟神奇寶貝gaole一樣的玩法，收集小朋友畫怪物
平台：Web網頁式
遊玩人數：1~2
客群：玩過gaole的玩家(全年齡)
獲利模式：廣告跟贊助

---

## 遊戲概念
透過時下流行的寶可夢gaole玩法，將精靈球替換成其他東西，精靈則收集小朋友畫的怪物畫，遊戲玩法大致一樣當作一個前端的練習跟風一波蹭流量。

---

## 預期目標
可以自行上傳小朋友畫像建立精靈，後期如有可愛熱門的可以發行周邊販售紀念或贊助贈紀念品。
